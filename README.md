# エイプリールフール企画  〜ITカレッジこれくしょん( アイコレ)〜

## 趣旨・目的

typn先生がやたら出てくる ~~艦これのパクリ~~ を作る。

## 内容

　建造の代わりにあざらしじゃんけんのようなものをして、勝利したら一定の確率でアイカレの先生が仲間に加わる。負けたらつよぽん先生が(勝利しても一定確率で)仲間に加わる。

建造と編成(?)くらいしか実装しない。 旗艦にした先生のボイスも・・・
